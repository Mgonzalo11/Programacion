</h1>
<hr style="border: 2px solid #4CAF50;">
<h2 class="code-line" data-line-start=4 data-line-end=5 style="color: #333;">
    <a id="_Descripcin_4"></a>📖 Descripción
</h2>
<p class="has-line-data" data-line-start="6" data-line-end="7" style="font-size: 1.2em; line-height: 1.5;">
    El <strong>Sistema de Gestión de Habitantes del Padrón</strong> es una aplicación interactiva diseñada para simplificar la gestión de datos de ciudadanos en registros municipales. Esta herramienta permite a los usuarios:
</p>
<ul style="list-style-type: square; padding-left: 20px;">
    <li class="has-line-data" data-line-start="8" data-line-end="9"><strong>Añadir nuevos habitantes</strong> al padrón.</li>
    <li class="has-line-data" data-line-start="9" data-line-end="10"><strong>Visualizar la lista de habitantes</strong> de manera clara y organizada.</li>
    <li class="has-line-data" data-line-start="10" data-line-end="12"><strong>Recibir ayuda</strong> sobre los comandos disponibles en la aplicación.</li>
</ul>
<p class="has-line-data" data-line-start="12" data-line-end="13" style="font-style: italic;">
    Esta aplicación es ideal para oficinas municipales, registros civiles o cualquier entidad que necesite mantener un registro accesible de los ciudadanos.
</p>
